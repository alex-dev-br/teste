unauthorized

HTTP 401 Handler: BadCredentialsException

Definição do tipo (estável)

Type (URI)
http://localhost:8081/erros/unauthorized/
Title (recomendado)
Authentication failed (ou traduza conforme seu padrão)
Quando usar
Credenciais inválidas ou ausentes.
Extensões
reason (apenas em dev), timestamp, instance

Exemplo de resposta (dinâmica)

{
        "type": "http://localhost:8081/erros/unauthorized",
        "title": "Falha ao autenticar",
        "status": 401,
        "detail": "Falha ao autenticar o usuário, verifique as credenciais e tente novamente.",
        "instance": "/login",
        "timestamp": "2025-10-27T12:34:56Z",
        "reason": "Bad credentials"
        }

Contrato

CampoTipoObrigatórioSemântica
type string (URI) Sim Identificador do tipo.
title string Sim Título canônico.
status number Sim HTTP status.
detail string Não Dinâmico.
instance string Não Dinâmico.
timestamp string Não Dinâmico.
reason string Não Evite em produção; útil em dev.
