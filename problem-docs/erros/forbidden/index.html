forbidden

HTTP 403 Handler: AuthorizationDeniedException

Definição do tipo (estável)

Type (URI)
http://localhost:8081/erros/forbidden/
Title (recomendado)
Operação não permitida
Quando usar
Usuário autenticado (token válido), mas sem permissão para executar a ação solicitada (papel/regra insuficiente).
Extensões
timestamp, cause (dev), instance

Exemplo de resposta (dinâmica)

{
        "type": "http://localhost:8081/erros/forbidden",
        "title": "Operação não permitida",
        "status": 403,
        "detail": "Operação não permitida, verifique as permissões de acesso.",
        "instance": "/api/v1/users",
        "timestamp": "2025-10-27T12:34:56Z"
        }
