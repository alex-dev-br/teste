Problem DetailsRFC 7807/9457

Tipos de erro (Problem Details)

Definição dos “problem types” usados pela API. Configure o campo type das respostas para apontar para estas páginas (ex.: http://localhost:8081/erros/<slug>/). Doc é estática> (tipo); as respostas da API> são dinâmicas (ocorrência).

invalid-data

Campos do corpo inválidos (Bean Validation + @Valid).

400 (sugerido) Handler: MethodArgumentNotValidException

invalid-parameter

Parâmetros de rota/consulta inválidos (@Validated).

400 (sugerido) Handler: ConstraintViolationException

malformed-json

Corpo ilegível/formato inválido.

400 (sugerido) Handler: HttpMessageNotReadableException

invalid-argument

Argumento inválido genérico.

400 (sugerido) Handler: IllegalArgumentException

unauthorized

Falha de autenticação.

401 Handler: BadCredentialsException

forbidden

Operação não permitida.

403 Handler: AuthorizationDeniedException

business-rule

Violação de regra de negócio.

422 Handler: BusinessValidationException

current-password-mismatch

Senha atual incorreta ao tentar alterar a senha.

422 Handler: CurrentPasswordMismatchException

internal

Erro inesperado.

500 Handler: Exception (fallback)
