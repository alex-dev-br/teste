business-rule

HTTP 422 Handler: BusinessValidationException

Definição do tipo (estável)

Type (URI)
http://localhost:8081/erros/business-rule/
Title (recomendado)
Business rule violation
Quando usar
Uma ou mais regras de negócio foram violadas.
Extensões
invalidParams[] ou errors[] (JSON Pointer)

Exemplo de resposta (dinâmica)

{
        "type": "http://localhost:8081/erros/business-rule",
        "title": "Violação de regra de negócio",
        "status": 422,
        "detail": "Uma ou mais regras de negócio foram violadas.",
        "instance": "/api/v1/users",
        "timestamp": "2025-10-27T12:34:56Z",
        "invalidParams": [{ "name": "age", "message": "must be ≥ 18" }],
        "errors": [{ "pointer": "/age", "detail": "must be ≥ 18" }]
        }

Contrato

CampoTipoObrigatórioSemântica
type string (URI) Sim Identificador do tipo.
title string Sim Título canônico.
status number Sim HTTP status.
detail string Não Dinâmico.
instance string Não Dinâmico.
timestamp string Não Dinâmico.
invalidParams array Não Extensão.
errors array Não Alternativa.
