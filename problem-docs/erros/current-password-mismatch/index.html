current-password-mismatch

HTTP 422 Handler: CurrentPasswordMismatchException

Definição do tipo (estável)

Type (URI)
http://localhost:8081/erros/current-password-mismatch/
Title (recomendado)
Senha atual incorreta
Quando usar
Ao tentar alterar a senha e o campo currentPassword não corresponde à senha vigente do usuário autenticado.
Extensões
timestamp, instance

Exemplo de resposta (dinâmica)

{
        "type": "http://localhost:8081/erros/current-password-mismatch",
        "title": "Senha atual incorreta",
        "status": 422,
        "detail": "A senha atual informada não confere.",
        "instance": "/api/v1/users/me/password",
        "timestamp": "2025-10-27T12:34:56Z"
        }

Contrato

CampoTipoObrigatórioSemântica
type string (URI) Sim Identificador do tipo.
title string Sim Título canônico.
status number Sim HTTP status.
detail string Não Dinâmico.
instance string Não Dinâmico.
timestamp string Não Dinâmico.
