internal

HTTP 500 Handler: Exception (fallback)

Definição do tipo (estável)

Type (URI)
http://localhost:8081/erros/internal/
Title (recomendado)
Internal server error
Quando usar
Erro inesperado não mapeado por handlers específicos.
Extensões
timestamp, instance, cause (dev)

Exemplo de resposta (dinâmica)

{
        "type": "http://localhost:8081/erros/internal",
        "title": "Internal server error",
        "status": 500,
        "detail": "An unexpected error occurred. Please contact support.",
        "instance": "/api/v1/users",
        "timestamp": "2025-10-27T12:34:56Z"
        }

Contrato

CampoTipoObrigatórioSemântica
type string (URI) Sim Identificador do tipo.
title string Sim Título canônico.
status number Sim HTTP status.
detail string Não Dinâmico.
instance string Não Dinâmico.
timestamp string Não Dinâmico.
cause string Não Somente dev; nunca exponha em produção.
